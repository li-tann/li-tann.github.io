复数影像的两种表现形式​

复数影像的数字存储形式

a+bia+bia+bi

复数影像的强度相位表现形式

u=∣u∣ejϕu=\lvert u\rvert e^{j\phi}u=∣u∣ejϕ

其中， ∣u∣\lvert u\rvert∣u∣为强度, ϕ\phiϕ为相位。

两种表达方式的相互关系为：

∣u∣=u.real2+u.imag2=a2+b2\lvert u\rvert={u.real}^2+{u.imag}^2=a^2+b^2∣u∣=u.real2+u.imag2=a2+b2
ϕ=arctan⁡u.imagu.real=arctan⁡ba\phi=\arctan\frac{u.imag}{u.real}=\arctan\frac{b}{a}ϕ=arctanu.realu.imag​=arctanab​
